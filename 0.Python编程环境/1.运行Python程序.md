## 运行python文件
```python
#! /usr/bin/python3

sum = 0
for i in range(101):
  sum = sum + i

print("The sum of [1,...,100] is: ")
print(sum)
```

- 第一种运行方式

  `python 文件名`

- 第二种运行方式

  文件首行增加执行程序：`#! /usr/bin/python`

  增加文件可执行属性：`chmod +x 文件名`：

  *注：在Windows下编辑完python文件即可直接运行（只要后缀名为.py)*

- 第三种方式（在Python程序中运行）
```python
import os
os.system('./test.py')
```

- 第四种方式（在Python程序中运行）
```python
import test

import sys
del sys.modules['test']
```
